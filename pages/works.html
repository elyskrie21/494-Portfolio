MoonDeal

Description of Project

Moondeal is a dynamic Chrome extension and website designed to help online shoppers track product prices from Amazon, Target, eBay, and more. It notifies users when prices drop, solving the issue of missing out on deals.

My Role

I was the lead front-end developer and contributed to designing the user experience for both the Chrome extension and the website. I also implemented backend API logic and assisted with deployment.

  4. REST API Development
  5. Server Hosting

Pantry

Description of Project

Pantry is a user-friendly Android application created to streamline collaborative meal planning for groups like families and roommates. The app allows users to propose meals, vote on options, create grocery lists, and manage shared meal responsibilities with ease.

My Role

I served as the lead backend architect, responsible for designing and building the RESTful API using Spring Boot. I handled server-side logic, data modeling, and integration with the MySQL database, ensuring secure and efficient communication between the Android client and backend services.

Skills or Knowledge Gained

  4. Authentication and session management
  5. Collaborative software planning and version control (Git)

Resources Used

Sketch-to-Image Translation with GANs

Description of Project

This project explores the use of Generative Adversarial Networks (GANs), specifically the Pix2Pix framework, to translate hand-drawn sketches into photorealistic images. The model was trained on paired image data, using a U-Net-based generator and PatchGAN discriminator, with a goal of improving creative workflows in areas like design and digital art.

My Role

I co-led the project, contributing to both the model architecture and backend implementation in PyTorch. My focus included redesigning the generator and discriminator architectures, improving data preprocessing, and enhancing training stability using advanced loss functions and augmentation strategies.

Skills or Knowledge Gained

  1. Deep learning with PyTorch
  2. GAN architecture (U-Net Generator, PatchGAN Discriminator)
  3. Training strategies (adversarial + L1 loss, label smoothing)
  4. Image preprocessing and augmentation
  5. Evaluation with FID score and visual inspection

Resources Used

  • Pix2Pix GAN framework
  • Sketchy Dataset for paired image training
  • NVIDIA RTX 3060 GPU for model training
  • Python, PyTorch, Jupyter Notebook
  • FID scoring tools for quantitative analysis
